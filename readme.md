# 项目介绍
hey-go-zero是基于[go-zero](https://github.com/tal-tech/go-zero) 的一个示例工程，通过一个简单的`选课`
系统来介绍一个项目的基本过程：
* 需求评估
* 研发
* 测试
* 部署
* 日志收集
* 指标监控
* 报警

除此之外，我们还将介绍go-zero的工具goctl的使用，如：
* 环境依赖
* 安装
* 命令介绍
* 基本使用
